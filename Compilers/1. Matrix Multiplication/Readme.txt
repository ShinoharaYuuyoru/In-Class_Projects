1. /bin文件夹：为*.exe程序（仅C++和Haskell）；在Java为Main.class；在Python为*.py。
2. /src文件夹：C++中为*.cpp；Java为*.java；Python为*.py；Haskell为*.hs。
3. /doc文件夹：输出。例：16_16.txt，输出为*.16.txt，“*”号为编号，以C++、Java、Python、Haskell依次从1编号。
4. PNG文件夹：截图文件夹。
5. Test_Example：测试用例文件夹，有16_16.txt，256_256.txt，1024_1024.txt。各为两个16*16矩阵、两个256*256矩阵、两个1024*1024矩阵相乘。
5. 测试用例均相同。输出时矩阵的格式无差别，提示语句的输出有格式差别。
